# Changelog

All notable changes to QR Studio are documented here.

---

## [2.0.0] — 2026-05-15 — Major Release

### Added
- Brand-new Material You design with dynamic colors on Android 12+.
- Custom themes — choose from a curated set of accent colors.
- Custom logo overlay for generated QR codes.
- Batch QR code generation — generate multiple codes from a CSV or list.
- PDF417 barcode support.
- In-app code preview before saving or sharing.
- Multi-language support (English, Spanish, French, German, Japanese, Chinese).

### Improved
- Up to 40% faster scanning on mid-range devices.
- Reduced app size by 25%.
- Lower battery consumption during long scanning sessions.
- Redesigned history list with quick actions and filters.
- Improved dark mode with better contrast.

### Fixed
- Fixed an issue where Wi-Fi QR codes would not connect on some Android 13 devices.
- Fixed a crash when generating very long URLs (>2000 characters).
- Fixed incorrect timestamps in the history list in certain timezones.
- Resolved a memory leak during continuous scanning.
- Fixed layout issues on tablets in landscape orientation.

---

## [1.1.0] — 2025-09-20

### Added
- Dark mode with automatic switching based on system theme.
- Searchable scan history.
- Export scan history as CSV.
- Share generated codes directly to any app.
- Customizable error-correction level for generated QR codes.
- Haptic feedback on successful scan (toggle in Settings).

### Improved
- Refreshed app icon and splash screen.
- Smoother transitions and animations.
- Reduced cold-start time by ~30%.
- Improved help tooltips throughout the app.

### Fixed
- Fixed an issue where the scanner would freeze on certain Samsung devices.
- Fixed incorrect QR code color rendering in dark mode.
- Fixed a crash when saving a code to the photo library on Android 11.
- Resolved an issue where the app would not restore the last-used code type on relaunch.

---

## [1.0.0] — 2025-03-10 — Initial Release

- QR Code Scanner — Fast, accurate scanning of all standard QR codes.
- QR Code Generator — Create QR codes for URLs, text, email, phone, SMS, contact (vCard), Wi-Fi, and location.
- Barcode Generator — Generate 1D barcodes (EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, Code 93, ITF, Codabar).
- Scan History — Automatically save every scan locally.
- Save & Share — Save generated codes as PNG images, or share them via any app.
- Offline Functionality — Core features work without an internet connection.
- Clean, modern interface.
- No account required. No ads. No tracking.
